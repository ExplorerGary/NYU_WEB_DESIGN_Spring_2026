Project Description

This webpage is designed as a memorial and interactive exploration page for BanG Dream!, aiming to showcase its history, music, and community through a visually engaging layout.

Overall Structure

The webpage is called Hoshi no Kodo, which means "Star Beats." It is a memorial page dedicated to BanG Dream, a multimedia project that includes music, anime, and games. The design consists of five major components organized into four layers.

1. Hero Section (Header Banner)

This section includes a main title, subtitle, and a call-to-action button placed at the center of the page. The button is designed with a bright color to draw attention and guide users to the External Link section in the same page, encouraging further exploration of the BanG Dream universe.

2. Navigation & Toolbars

The navigation bar includes a themed logo and three main tabs: Home, Bands, and Library. These allow users to navigate between different sections of the website.

A secondary contact toolbar provides access to personal information and external platforms (email, Instagram, GitHub). In the mobile version, this section is repositioned to improve usability.

3. Main Content – Timeline ("Starry Path")

The core feature of this design is a vertical timeline that presents key milestones in BanG Dream’s history. It uses a "fishbone" layout combined with a starry visual theme to create a sense of journey and storytelling.

The timeline is designed to be scrollable, with content gradually revealed using fade-in effects, enhancing user engagement and the feeling of discovery.

The Milestone is organized in a structure called StoryItem, consisting of a summary, a image, and a star icon. The user can click on the images to view more details about each milestone, which will be displayed in a pop-up modal with additional information and media (if I figured out how to implement it).

4. External Link Footer

This section contains clickable images linking to official BanG Dream content, namely the games.

Design Considerations

The design emphasizes bright gradients, soft shadows, and rounded elements to reflect the energetic and emotional tone of BanG Dream. The layout adapts between desktop and mobile versions to ensure accessibility and consistency across devices. It serves as the final call-to-action area, encouraging users to continue exploring beyond this page. It is worth noting that slight adjustments are made to inprove the readability of the text and the visibility of the contents, especially in the mobile version, where the layout is modified to ensure a better user experience.
